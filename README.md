# Automation Exercise — E2E Test Suite

Test automation project for [automationexercise.com](https://automationexercise.com), covering UI flows and REST API validation using Playwright.

## Stack
- [Playwright](https://playwright.dev/) — E2E testing
- [Faker.js](https://fakerjs.dev/) — test data generation
- [dotenv](https://github.com/motdotla/dotenv) — environment config

## Setup

```bash
npm install
npx playwright install
```

## Running tests

```bash
npx playwright test
```

## Environment

Copy `automation.env` and fill in your credentials before running.
